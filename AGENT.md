# 俄勒冈之旅 TUI 开发任务

## 项目概述
复刻经典游戏《The Oregon Trail》(俄勒冈之旅)，使用 Go + Bubble Tea 框架开发 TUI 版本。

## 游戏背景
1971年经典教育游戏，模拟1848年美国移民从密苏里州独立城到俄勒冈州威拉米特谷的2000英里旅程。

## 核心功能需求

### 1. 角色创建系统
- 选择职业：银行家($1600)、木匠($800)、农夫($400)
- 命名家庭成员（最多5人）

### 2. 商店购买系统
- 牛(oxen)：拉车必需，$20/对
- 食物(food)：每日消耗，$0.20/磅
- 衣服(clothing)：御寒防病，$10/套
- 弹药(ammunition)：狩猎用，$2/20发
- 零件(spare parts)：修车用，$10/个

### 3. 旅程系统
- 里程碑地点：独立城→堪萨斯河→蓝河→福特要塞→拉勒米要塞→独立岩→南关→福特布里杰→蛇河→蓝山→达尔斯→俄勒冈城
- 每日行进（15-40英里，取决于速度设置）
- 天气变化（晴、雨、雪、暴风雪、酷热、大雾）
- 季节变化（3月出发，经历春夏秋冬）

### 4. 随机事件系统
- 疾病：痢疾、伤寒、霍乱、麻疹、发烧、蛇咬
- 意外：断腿、中暑、溺水
- 盗贼袭击
- 牛死亡
- 车辆损坏
- 找到浆果/野果

### 5. 河流穿越
- 选择：渡船($5)、游泳(危险)、等待、雇佣向导($3)
- 根据河流深度和天气决定成功率

### 6. 狩猎小游戏
- 消耗弹药
- 随机遇到野牛、鹿、熊、兔子
- 射击成功获得食物

### 7. 资源管理
- 食物每日消耗（每人2-6磅，取决于配给设置）
- 牛的健康影响速度
- 衣服影响疾病概率
- 零件用于修车

### 8. 游戏状态
- 行进速度：稳健(20英里)、努力(30英里)、拼命(40英里)
- 食物配给：充足(6磅)、节约(4磅)、极简(2磅)
- 休息恢复健康

### 9. 胜利与失败
- 全员死亡 → 游戏结束
- 到达俄勒冈城 → 胜利
- 最终评分基于：存活人数、剩余资源、到达时间、职业

└── ui/
    ├── styles.go    # 样式定义（已创建）
    └── screens.go   # 界面逻辑（缺失！需要创建）

### 需要完成的工作
1. 检查并修复 game/ 目录下所有文件的编译错误
2. 创建 ui/screens.go - 实现所有游戏界面
3. 确保 main.go 正确连接所有模块
4. 测试游戏可运行

### UI 界面需求
- 主菜单界面
- 职业选择界面
- 家庭成员命名界面
- 商店购买界面
- 旅程主界面（显示状态、地图、操作菜单）
- 河流穿越界面
- 狩猎界面
- 事件处理界面
- 死亡界面
- 胜利界面

### TUI 风格
- 复古绿色终端风格
- ASCII 边框
- 清晰的菜单导航
- 中文界面

## 开发步骤
1. 先检查现有代码，修复编译错误
2. 实现 ui/screens.go
3. 运行 `go run main.go` 测试
4. 修复任何运行时问题
5. 确保所有游戏功能可用

## 注意事项
- 使用 `go env -w GOPROXY=https://goproxy.cn,direct` 设置代理
- 用户是中国人，界面使用中文
- 游戏要有完整可玩性，不是演示版
